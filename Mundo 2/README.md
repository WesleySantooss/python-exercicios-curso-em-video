# Mundo 2 – Estruturas de Repetição

Este diretório contém exercícios do Mundo 2 do curso de Python do Curso em Vídeo.

## Conteúdos abordados

* Estruturas de repetição (for e while)
* Condições compostas
* Contadores e acumuladores
* Lógica de programação aplicada

## Organização dos arquivos

Os exercícios estão nomeados com número e descrição para facilitar a identificação, por exemplo:

* Ex049 - Tabuada v2.0.py

## Progresso

Exercícios sendo resolvidos e adicionados gradualmente.
