# Mundo 1 – Fundamentos de Python

Este diretório contém exercícios iniciais de programação em Python.

## Conteúdos abordados

* Entrada e saída de dados
* Operações aritméticas
* Manipulação de strings
* Estruturas condicionais (if/else)

## Organização dos arquivos

Os exercícios estão nomeados com número e descrição para facilitar a identificação, por exemplo:

* Ex001 - Olá Mundo.py

## Quantidade de exercícios

35 exercícios resolvidos.
